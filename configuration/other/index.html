SeAT sources information about the SDE from a json file hosted here. It may happen that the SDE gets updated but the the json resource has not yet been updated. For this reason, its possible to specify the version to get based on what is available on https://www.fuzzwork.co.uk/dump/.

overriding the resources json

Check the version of SDE dumps available on https://www.fuzzwork.co.uk/dump/. At the time of this writing, frostline-1.0-116241 was the latest. Once you have the version string ready, open the .env configuration file and add a key as follows:

SDE_VERSION=frostline-1.0-116241

When running the SDE updater, specify the --local parameter to source the version string from the configuration file:

$ php artisan eve:update-sde --local

notes

By default, SeAT automatically updates the SDE every month. You may want to login as an administrator and remove the schedule to update it monthly if you have overridden the default.
